 Navegação via command line interface e instalação 
    - Comandos básicos para um bom desempenho no terminal 
  O que vamos aprender:

- Mudar de pastas
- Listar as pastas
- Criar pastas/arquivos
- Deletar pastas/arquivos

Digitando cd / no prompt (CMD) ele entra na pasta C
Digitando cd Windows no prompt (CMD) ele entra na pasta Windows

 - dir traz todos os arquivos, programas, pastas, etc contigos na pasta especifica.

 - cd .. retorcede o passo dado.

 - cls limpa o terminal (cmd) prompt

 - TAB auto completa a entrada.
   Exemplo:
   Se no prompt c:> colocarmos cd W, apertando TAB ele completando o restante 
   ficando c:> cd Windows

Como criar uma pasta no prompt.
 Usa-se o comando c:> mkdir (mais o nome) workspace.
Assim se criou a pasta workspace
 Apertando o enter se nada acontecer ou ser apresentado é sinal que a criação deu certo.

  Para criar arquivo usa-se echo hello (nome do arquivo) > (redirecionador) hello.txt (nome arquivo mias extensão)

  Para deletar conteudo da pasta usa-se o del mais neme do arquivo
Ex:
C:> del workspace
 aparecerá Tem certeza (S/N)?

  Para deletar o repositorio usa-se o rmdir mais nome do arquivo e as "flags" /S /Q
Ex:
C:> rmdir workspace /S /Q
